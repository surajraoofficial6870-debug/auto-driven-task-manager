# 🚀 Team Task Manager (Full-Stack)

A professional web application built for seamless team collaboration, project management, and task tracking with **Role-Based Access Control (RBAC)**.

### 🌟 Key Features
*   **Authentication & Security:** Secure Signup/Login system utilizing JWT tokens and Bcrypt password hashing.
*   **Project Management:** Admin-only capability to create, view, and manage multiple projects.
*   **Task Assignment:** Dynamic task creation with assignment to specific team members.
*   **Real-time Dashboard:** Summary of total tasks, completion status, and overdue task alerts.
*   **Role-Based Access (RBAC):** 
    *   **Admin:** Full access to manage projects, members, and all tasks.
    *   **Member:** View assigned tasks and update status (Pending/Completed).

### 🛠️ Tech Stack
- **Backend:** Flask (Python), Flask-SQLAlchemy (SQLite)
- **Security:** JWT (JSON Web Tokens), Flask-CORS
- **Frontend:** Vanilla JavaScript, HTML5, CSS3
- **Deployment:** Railway
